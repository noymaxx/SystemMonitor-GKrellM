# MyGKrellm

System monitor inspired by GKrellM.

## Build

```bash
make
```

## Usage

```bash
./MyGKrellm -t    # Text mode (nCurses)
./MyGKrellm -g    # Graphical mode (SFML)
./MyGKrellm -h    # Help
```

## Features

- Hostname and username
- OS and kernel version
- Date and time
- RAM usage
- Battery status
- Network stats

## Dependencies

- ncurses
- SFML 2.5+
